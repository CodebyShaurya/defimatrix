# About DeFiMatrix

DeFiMatrix is revolutionizing decentralized finance through AI-powered automation and intent-driven execution. Our platform requires the highest level of security to protect user funds and ensure the integrity of automated operations across multiple protocols and chains.

## Role Overview

As a Security Engineer and DeFi Protocol Auditor, you'll be responsible for ensuring the security and reliability of our smart contract systems and protocol integrations. You'll work on auditing code, implementing security measures, and developing robust testing frameworks.

## Key Responsibilities

- Conduct comprehensive security audits of smart contracts
- Design and implement security testing frameworks
- Review protocol integrations for security vulnerabilities
- Develop automated security monitoring systems
- Implement security best practices and standards
- Create security documentation and guidelines
- Perform threat modeling and risk assessments
- Collaborate with blockchain and AI teams on security
- Monitor and respond to security incidents
- Research emerging security threats and mitigations

## Required Skills & Experience

- 5+ years of security engineering experience
- Strong experience with smart contract auditing
- Deep understanding of DeFi security risks
- Expertise in blockchain security best practices
- Experience with security tools and frameworks
- Strong knowledge of cryptography
- Excellent problem-solving abilities
- Track record of identifying vulnerabilities

## Nice to Have

- Experience with formal verification
- Knowledge of AI/ML security
- Background in traditional security
- Contributions to security research
- Experience with cross-chain security
- Understanding of MEV and front-running
- Security certifications (CISSP, CEH, etc.)

## What We Offer

- Competitive salary and equity package

- Remote-first work environment with flexible hours
- Comprehensive health and wellness benefits
- Professional development and conference budget
- Access to cutting-edge security tools
- Collaborative and innovative team culture
